Projects

  • Streaming on AWS with Flink: A streaming pipeline that replicates a music streaming platform, all on AWS resources. It uses Apache Kafka, Apache Flink on Amazon EMR. Redshift is used as the warehouse and Metabase is used as the BI tool.

  • Web Scraping DrugAge: An ETL pipeline scraping data from drug age using Dagster, duckDB, dbt, BigQuery, and Streamlit. See the live app here (hosted in Azure with Docker)

  • Datawarehouse Management App: An app for writing data to Google BigQuery. I have frequently encountered clients that need to write data back at the data warehouse level. I have successfully addressed this challenge using various technologies, including Streamlit, PowerApps, Django, Excel, and SharePoint. In this project, I leverage FastAPI and AgGrid, which strike a great balance between simplicity (compared to Django) and front-end control (compared to Streamlit).
